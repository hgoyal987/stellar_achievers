# 🚀 Stellar Achievers

> Fast-Track Your Global Career & Education. Expert coaching for PTE, IELTS, OET, and NAATI.
> A fast, static lead-generation landing page for educational coaching with direct WhatsApp lead routing.

## ✨ Features

* **Instant Communication:** Direct-to-WhatsApp form submission without needing a backend database.
* **Responsive Design:** Fully mobile-optimized layout.
* **Lightweight:** Zero-dependency static architecture for lightning-fast load times.

## 🛠️ Tech Stack

* **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript
* **Hosting:** Netlify / GitHub Pages
* **Integrations:** WhatsApp `wa.me` API
